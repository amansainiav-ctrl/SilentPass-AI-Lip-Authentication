# SilentPass: AI-Based Silent Lip Movement Authentication System

## Project Description
SilentPass is an AI-based authentication system that verifies a user using silent lip movement patterns instead of passwords or voice commands. The system uses computer vision and machine learning to analyze lip motion and compare it with a stored authentication pattern.

## Problem
Traditional authentication methods such as passwords and PINs are vulnerable to:
- Phishing attacks
- Shoulder surfing
- Password leaks

## Proposed Solution
SilentPass allows users to authenticate silently by speaking a passphrase without sound.  
A camera captures the lip movement and an AI model verifies the pattern.

## Technologies Used
- Python
- OpenCV / MediaPipe
- Machine Learning (CNN / LSTM)
- Computer Vision

## System Workflow
Camera → Face Detection → Lip Detection → AI Model → Authentication Result

## Applications
- Smart Door Locks
- Secure Workstation Login
- ATM Authentication
- Smartphone Unlock
